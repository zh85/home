### 第14页 — 晶体关键发现

**标题**: 晶体生成: 系统消融实验与关键发现

**内容** (紧凑表格 + 三条核心结论):
```
7方法×5指标对比 (MP-20, n≈10000):

方法              valid    COV-R    密度wdist
LLM-Only          79.92   96.75     0.98
Unconditional     86.22   99.15     0.10      ← 最佳LLM-based
Additive          86.22   53.54     3.56      ← 崩溃
Cross-Attn        85.75   94.43     1.51
GT comp+rand      91.61   99.83     0.60      ← 扩散理论上限

三条核心发现:
① 无条件扩散最优 — LLM条件注入成为扩散去噪的噪声
  → LLM价值在Stage1(生成初始结构), 不在Stage2(引导去噪)

② GT组分消融: 扩散上限(91.61%) >> LLM-based(86.22%)
  → LLM是当前系统的瓶颈，尤其在组分预测

③ 最优路径: 解耦LLM(组分)和扩散(结构)
  → 指导后续工作: 让各自做最擅长的事
```
