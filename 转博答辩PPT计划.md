### 第14页 — 晶体生成: 系统消融实验

**标题**: 基于CrysLLMGen的系统消融 (7方法×5指标)

**内容** (核心数据):
| 方法 | struct_valid | valid | COV-R | 密度 |
|------|:---:|:---:|:---:|:---:|
| LLM-Only | 92.08 | 79.92 | 96.75 | 0.98 |
| Unconditional | **99.93** | **86.22** | **99.15** | **0.10** |
| Additive | **99.94** | **86.22** | 53.54 | 3.56 |
| Cross-Attn | 99.29 | 85.75 | 94.43 | 1.51 |
| **GT comp+rand** | 99.69 | **91.61** | **99.83** | 0.60 |

**三个关键发现**:
1. 无条件扩散最优 — LLM条件注入反而降低扩散性能
2. GT组分消融: 扩散上限 (valid=91.61%) >> LLM-based (86.22%) — **LLM是瓶颈**
3. 最优路径: 解耦 — LLM只管组分, 扩散只管结构

**视觉**: 热力图

### 第18页 — 多尺度整合: 技术路线

**标题**: 多尺度过程知识图谱

**技术路线**:
```
Step 1: 信息提取
  PFD → 图(GNN): 设备节点 + 流股边
  P&ID → 图(GNN): 设备+仪表+控制节点
  文献/专利 → LLM提取: 反应条件+动力学参数

Step 2: 对齐与融合
  设备对齐: PFD中的"反应器R-101" = P&ID中的"R-101"
  条件关联: 反应网络的T,P → P&ID的温度压力控制回路
  尺度桥接: 反应动力学 → 反应器尺寸 → PFD设备参数

Step 3: 统一查询
  "提高温度对产品收率有什么影响?"
  → 检索反应网络(活化能) → 检索PFD(换热网络)
  → 推理: 温度↑ → 反应速率↑ + 副反应↑ → 收率先升后降
```
