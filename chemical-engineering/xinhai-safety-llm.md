- **投资**: **8500万元**
- **定位**: 全国首个化工安全领域垂直大模型 (已通过国家备案)
- **规模**: 规划22个应用场景, 已落地 **13个**

## 液化气智能装车场景

### 技术架构

| 组件 | 功能 |
|------|------|
| 高精度摄像头 | 捕捉静电接地、鹤管定位、人员操作等关键环节 |
| 视觉大模型 / AI视频分析 | 实时识别操作流程中的违规操作 |
| 自动鹤管定位 | AI视觉识别自动锁定注入口, 指挥鹤管精准移位插入 |
| 语音预警系统 | 实时语音告警 + 二次核验确认 |

### 实际案例 (2026年3月17日)

液化气装车区3号台位:
1. 摄像头捕捉到静电接地线与槽罐车接地端子连接松动
2. 触发语音预警: "请注意, 静电接地线连接不牢固, 请重新检查并固定"
3. 工作人员排查重新接地
4. 摄像头二次核验通过
5. 语音提示"静电接地合格, 可以继续作业"
6. **全程仅用2分钟**

### 硫酸智能装车 (类似场景)

- 98%浓硫酸装车自动化
