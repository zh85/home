# 陶氏化学张家港 — AI 工艺控制与数字化转型

## 基本信息

- **基地**: 陶氏化学张家港生产基地
- **运营**: 自 2002 年起, 亚太区最大生产基地之一
- **最新扩建**: 2025年3月有机硅下游产品扩建项目正式投产

## AI与数字化应用

| 方向 | 内容 |
|------|------|
| 生产能耗监控 | 实时能耗数据采集与优化 |
| 物料精准配送 | 自动化物料管理与配送系统 |
| 工艺/安全/环保监控 | 实时信息监控与预警 |
| 工业操作系统 | 基于 **蓝卓 supOS** 的智能车间建设 |
| 工业云平台 | 接入 **腾讯云 (张家港) 工业云平台** |
| 塑料回收追溯 | AI + 智能标签, 全生命周期追溯平台 (2025年4月联合发布) |

## 战略方向

陶氏大中华区总裁黄映雪:
- 加大数字化投入, 提升响应速度与服务能力
- 持续推动生产基地的智能化改造

## 说明

目前公开资料中尚未找到陶氏张家港工厂独立命名的"AI工艺控制系统"详细技术文档。其AI应用更多体现为:
- 基于 supOS 的智能车间建设
- 能耗与工艺参数的实时监控优化
- 工业云平台集成
